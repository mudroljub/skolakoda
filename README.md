# Sajt Škole koda

Vidi [skolakoda.org](https://skolakoda.org/)

## Instalacija

Prvo instaliraš [ruby](https://rubyinstaller.org/downloads/). Potom:

```
gem install bundler
gem install jekyll
```

## Razvoj

```
bundle install
bundle exec jekyll serve
```

Ili sa parametrima:
```
bundle exec jekyll serve --drafts --incremental
```
